=====
Muses
=====
Proof of concept frontend

Installation
============

.. code-block:: sh

    yarn install

    yarn start

Configuration
============
In order to connect to the API add the API url in the `shared/constats` file

.. code-block:: js

  const constants = {
    api: {
      url: '' //API URL
    }
  }
